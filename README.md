# Institute of Digital Risk Website

This project is a responsive single-page website created for the **Institute of Digital Risk (IDR)** using **HTML, CSS, and JavaScript**.

## Project Overview
The website introduces the mission and activities of the Institute of Digital Risk, which focuses on training and deploying professionals in digital, cyber, and technology risk management.

## Features
- Hero section highlighting IDR's mission
- About section explaining the institute
- Service pillars including:
  - Academy
  - Innovation & Incubation
  - Advisory Services
- Community section describing who IDR serves
- Contact form for registering interest
- Sticky navigation bar
- Responsive layout for mobile and desktop devices

## Technologies Used
- HTML5
- CSS3 (Flexbox and Grid)
- JavaScript
